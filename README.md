# JavaScript For Loop Programs 🔁

This repository contains basic JavaScript programs using **for loop**.

## 📁 Programs Included

- Even Numbers
- Odd Numbers
- Factorial
- Factors of a Number
- Basic Loop Examples (ex1, ex3, ex4)

## 💡 Technologies Used

- HTML
- JavaScript

## 🚀 How to Run

1. Download or clone the repository
2. Open any `.html` file in browser
3. Check output in browser console or screen

## 📌 Purpose

This project is created to practice and understand the working of **for loop in JavaScript**.
